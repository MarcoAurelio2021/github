Inicio

Criando uma pasta

  • cd Doc (aperta table)
  • mkdir + nome da pasta
  • cd + nome da pasta
  • git init
  • code . (para abrir vs code)

Adicionar e comitar

  • git add .
  • git commit -m "nome"

Enviar para git hub

  • git remote add...(para fazer a ligação do git hub)
  • git push + o link de envio

  • Utilidades

    • git pull - para trazer alterações da nuvem.
    • git clone - para trazer um repositorio
    • git log - para avaliar as alterações de um arquivo.
    • vim - para navegar e manipular um texto.
    • git status - para ver as etapas.
    • rmdir - deletar.
    • rm - deletar por nome.
    • ls - al .git - para fazer uma listagem.
